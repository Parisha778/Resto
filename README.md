# 🍽️ Projet Resto - Site Vitrine

Ce projet est un site vitrine moderne pour un restaurant, réalisé dans le cadre de mon apprentissage du développement web. L'objectif était de créer une interface fluide, responsive et attrayante pour présenter un menu et l'ambiance d'un établissement.

## 🛠️ Technologies utilisées
* **HTML5** : Structure sémantique du site.
* **CSS3** : Design personnalisé et mise en page.
* **Bootstrap 5** : Utilisation du framework pour le responsive (grille) et les composants (navbar, boutons).
* **Google Fonts** : Intégration de polices typographiques élégantes.

## 🌟 Fonctionnalités
* **Design Responsive** : Adapté aux mobiles, tablettes et ordinateurs.
